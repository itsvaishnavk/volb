About

VOLB allows you to control your device volume without using physical buttons.
It creates floating on-screen controls that work over any app.
Ideal for devices with broken volume keys or for users who prefer touch controls.

Features

Floating draggable volume buttons
Auto-hide and transparency after inactivity
Snap to screen edges
Custom size, spacing, and position
Quick remove via drag-to-delete

Permissions

Display Over Apps – Required for floating buttons
Foreground Service – Keeps controls active
Internet – Used only for optional in-app popup messages
VOLB does NOT access contacts, files, camera, or personal data.

Privacy

No data collection
No tracking or analytics
No data sharing

Technical Details

Uses Android WindowManager overlay system
Foreground service ensures continuous operation
Real-time updates using SharedPreferences listener
Optional popup system using API (on app launch only)
